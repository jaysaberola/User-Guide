Telephone Directory – User Guide

Philsaga Intranet  ·  WebFocus Solutions, Inc.  ·  February 2026

How to use this guide: Click any question below to reveal the answer with screenshots from the official user guide. This guide covers everything you need to search, export, and update contact information in the Philsaga Intranet Telephone Directory.
01

Introduction

The Telephone Directory is a Search Engine built into the Philsaga Intranet. It allows all internal staff to quickly look up emergency contact numbers, local extensions, and office locations.

Slide 1 – Introduction

All internal staff members of Philsaga Mining Corporation (PMC) who need to search for contact information quickly within the company intranet.

02

Accessing the System

Open any web browser and navigate to:

https://intranet.philsaga.com/telno/sengine.php

Slide 2 – Intended Audience & Access

Tip: Bookmark this URL for faster access in the future.

You will see the main Telephone Directory interface with a Search Anything bar, a Show All button, and a Dialing Format panel on the right showing the most frequently used numbers.

03

Search Anything

  1. Click on the Search bar (labeled "Search Anything").
  2. Type a keyword — you can search by Name, Local number, or Location.
  3. Press Enter on your keyboard.
  4. Results will appear below, showing the contact's Name, Local, and Location.

Slide 3 – Manage Search Anything

Example: Type HR MANAGER and press Enter to find all HR Manager entries.

You can search using any of the following:

  • Employee Name / Position Name (e.g., "HR Manager", "Accounting Manager")
  • Local Number (e.g., "3135")
  • Location (e.g., "MILL", "MINES", "DAVAO")

Each result card displays: Name, Local (extension number), and Location. There is also a Request number update link in each card.

The right-side Dialing Format panel shows the most frequently used numbers (Mill Emergency Response Team, Mill Safety, Mine Safety, etc.). Simply click any item to instantly display its details — no need to type in the search bar.

To call via landline, use the format:

Dial # 082-234-0045 TO 47 + LOCAL

Dial the main trunk number, then add the local extension shown in the directory.

04

Export to Excel

  1. Perform a search — results must be displayed first.
  2. The Export to Excel (green) button will appear above the results.
  3. Click it to download the file in Excel format.
  4. Save the file to your local drive.

Slide 4 – Export Excel File

Note: The Export to Excel button only appears after search results are shown.

Yes. Click Show All first to load the complete contact list, then click Export to Excel to download all contacts in one file.

05

Show All

  1. Click the Show All button on the right side of the Search bar.
  2. The system will display all available telephone numbers with Name, Local, and Location.
  3. Scroll down to view the full list.

Slide 5 – Show All

The Show All button is located on the right side of the Search Anything bar at the top of the page. It is an orange button labeled "Show All".

06

Request Number Update

  1. Find the contact in the results list.
  2. Click the Request number update link below the contact's details.
  3. A Telephone Update Request Form will pop up.
  4. Fill in:
    • Reason for update request
    • Update details – the new/corrected local number
    • Requestor – your name
  5. Click Submit.

Slide 6 – Request Number Update

What happens next? The request is automatically sent to the IT Helpdesk via email for action.

After clicking Submit, the request is routed to the IT Helpdesk via email. The IT team will review and update the directory accordingly.

Yes. The form has a Close button beside Submit. Click Close to dismiss the form without sending any request.
